# mee6_bulk_badwords (Windows)
Adds and Removes bad words in Bulk. Designed only for Mee6.

# Dependencies
- python3.7
- selenium
- time

# Downloads Required
Download chromedriver based on your chrome version!
https://sites.google.com/a/chromium.org/chromedriver/downloads
